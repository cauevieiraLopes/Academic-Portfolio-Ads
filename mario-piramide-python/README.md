# 🧱 Pirâmide do Mario em Python (`CS50 - Mario`)

Este repositório contém a resolução em Python do clássico desafio **Mario** (Semana 1 do curso CS50 de Harvard). O objetivo do projeto é renderizar uma pirâmide alinhada à direita utilizando caracteres ASCII baseada em um tamanho fornecido pelo usuário.

## 📖 Descrição do Projeto
O programa solicita ao usuário a altura desejada para a pirâmide e a desenha no terminal. A renderização utiliza espaços em branco para o alinhamento correto dos blocos (`#`), simulando os obstáculos clássicos do jogo Super Mario Bros.

Exemplo de saída para tamanho **4**:
```text
   #
  ##
 ###
####
```

## 🚀 Como Executar
1. Certifique-se de ter o Python 3 instalado em sua máquina.
2. Clone este repositório ou navegue até a pasta do projeto.
3. Execute o arquivo via terminal:
   ```bash
   python main.py
