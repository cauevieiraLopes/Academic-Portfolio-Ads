# 🧮 Calculadora Interativa no Terminal (Python)

Este projeto consiste em uma calculadora de terminal desenvolvida em Python, com foco em controle de fluxo, tratamento de exceções e experiência do usuário (UX) em ambiente de linha de comando.

## 📖 Descrição do Projeto
O programa permite ao usuário realizar operações matemáticas básicas e avançadas (Soma, Subtração, Multiplicação, Divisão e Potenciação) de forma contínua através de um laço de repetição. 
O programa é meu primeiro código em busca de aprender a linguagem python.

## 🛠️ Recursos e Conceitos Aplicados
- **Tratamento de Exceções (`try / except`):** Captura de erros de tipo (`ValueError`) e erros matemáticos (`ZeroDivisionError`) para manter a aplicação estável.
- **Estruturas de Repetição e Decisão:** Uso de `while True` para execução contínua e estruturas condicionais (`if / elif / else`) para direcionamento das operações.
- **Tratamento de Strings:** Sanitização de entradas com métodos nativos do Python (`.strip()` para remover espaços extras e `.upper()` para padronização de caracteres).
- **Simulação de Delay:** Utilização do módulo nativo `time` para fornecer um feedback visual de encerramento do sistema.

## 🚀 Como Executar
1. Certifique-se de ter o Python 3 instalado em sua máquina.
2. Clone este repositório ou navegue até a pasta do projeto.
3. Execute o arquivo via terminal:
   ```bash
   python main.py
